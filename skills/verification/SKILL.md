# Verification Skill

## Objective

Validate whether outputs are:
- correct
- complete
- logically consistent
- aligned with requirements

---

# Rules

- Never assume correctness
- Verify claims against provided context
- Identify uncertainty explicitly
- Check for contradictions
- Validate edge cases

---

# Verification Workflow

1. Check requirement alignment
2. Identify unsupported assumptions
3. Validate logical consistency
4. Check missing scenarios
5. Evaluate ambiguity and risks
