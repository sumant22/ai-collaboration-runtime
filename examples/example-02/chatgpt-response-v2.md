# Task Response — ChatGPT v2 (Example 02)

## Task Reference
Plan how to organize customer follow-ups for a small retail shop.

## Skills Applied
- reasoning
- planning

---

## Analysis

### Known Facts
- Shop sells steel doors
- Owner handles sales with 1–2 staff
- Customer inquiries come from WhatsApp, phone calls, and walk-ins
- Follow-ups are sometimes forgotten
- No budget for paid software
- System must stay simple and quick to maintain
- Available tools: WhatsApp, Google Sheets, notebook

### Assumptions
- Staff can use WhatsApp daily
- At least one person has a smartphone with internet access
- Customer volume is manageable manually
- Most customers respond within a few days if interested

### Unknowns
- Exact number of inquiries per day
- Whether all staff can access Google Sheets
- Average time customers take before purchasing

---

## Proposed Approach

### Phase 1 — Create One Simple Follow-up Tracker (20 Minutes Setup)

Use either:
1. Google Sheets (recommended), OR
2. A paper notebook if internet access is unreliable

#### Google Sheet Columns

| Column | Purpose |
|--------|---------|
| Date | Inquiry date |
| Customer Name | Identify customer |
| Phone Number | Contact details |
| Inquiry Type | Door model or request |
| Source | WhatsApp / Call / Walk-in |
| Status | New / Follow-up / Interested / Closed / Lost |
| Next Follow-up Date | Next contact date |
| Last Action | Last message or call made |
| Owner/Staff | Who is responsible |

#### Example Tracker Row

| Date | Customer Name | Phone Number | Inquiry Type | Source | Status | Next Follow-up Date | Last Action | Owner/Staff |
|------|--------------|-------------|-------------|--------|--------|-------------------|------------|------------|
| 24-May | Rajesh Patil | 9876543210 | Steel Safety Door | WhatsApp | Follow-up | 27-May | Sent price photos | Amit |

---

### Phase 2 — Paper Notebook Format (Offline Backup)

If internet or Google Sheets is difficult, use a ruled notebook with these columns:

| Date | Name | Phone | Product | Follow-up Date | Status | Notes |
|------|------|-------|---------|---------------|--------|-------|

#### Example Notebook Entry
| 24-May | Rajesh Patil | 9876543210 | Steel Door | 27-May | Interested | Asked for price |

#### Notebook Rules
- One page per week
- Mark completed customers with a ✓
- Circle overdue follow-ups in red pen

---

### Phase 3 — Daily Follow-up Routine (10–15 Minutes Total)

#### Morning Routine (5–7 Minutes)

| Step | Action | Responsible |
|------|--------|------------|
| 1 | Open tracker/notebook | Staff |
| 2 | Check customers due today | Staff |
| 3 | Send WhatsApp or make quick call | Staff |
| 4 | Mark action completed | Staff |

#### Evening Routine (5–8 Minutes)

| Step | Action | Responsible |
|------|--------|------------|
| 1 | Add all new inquiries from today | Staff |
| 2 | Update customer status | Staff |
| 3 | Review overdue follow-ups | Owner |
| 4 | Confirm tomorrow's follow-ups | Owner |

---

### Phase 4 — Follow-up Timing Rules

| Situation | Action |
|-----------|--------|
| New inquiry received | Reply same day |
| No customer response | Follow up after 3 days |
| Still no response | Follow up again after 5 days |
| No response after 3 attempts | Mark as Lost |
| Interested customer | Follow up every 2–3 days until decision |

---

### Phase 5 — Overdue Follow-up Handling

An overdue follow-up means follow-up date passed with no action taken.

#### Handling Rule
1. Owner checks overdue entries every evening
2. Staff contacts overdue customers next morning first
3. Mark overdue items: RED color in Google Sheets OR red circle in notebook
4. No overdue item should remain pending more than 2 days

---

### Phase 6 — WhatsApp Follow-up Template

```
Hello [Customer Name],

This is [Your Name] from Next Generation Door.

Just checking whether you had any questions about the steel door you asked about earlier.

Please let us know if you would like pricing, photos, or installation details.

Thank you.
```

---

### Phase 7 — Responsibility Rules

#### Staff Responsibilities
- Add every inquiry immediately
- Send daily follow-ups
- Update customer status
- Mark completed actions

#### Owner Responsibilities
- Review tracker every evening
- Check overdue customers
- Ensure no inquiry is missed
- Handle high-value or serious buyers personally

---

## Risks

| Risk | Likelihood | Mitigation |
|------|-----------|------------|
| Staff forgets updates | Medium | Owner checks tracker daily |
| Duplicate entries | Medium | Search phone number before adding |
| Internet issues | Low | Use notebook backup |
| Too many pending follow-ups | Medium | Prioritize overdue customers first |
| Customers stop responding | Common | Stop after 3 follow-up attempts |

## Confidence Level
High — the process is intentionally simple, operationally realistic for a
small retail shop, and designed around existing tools and daily habits.

## Final Recommendation
Start with either Google Sheets + WhatsApp, OR Notebook + WhatsApp if staff
are not comfortable with Sheets. Begin using the tracker immediately for every
inquiry, even if customer volume is small. Consistency is more important than complexity.
